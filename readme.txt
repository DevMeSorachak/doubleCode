DoubleCode Desktop Application
------------------------------------
Open with Apache NetBeans
must have netbean more than version 12.0 ++++
	***** 
		1. UNZIP doublecode
		2. open Apache NetBeans IDE
		3. File > Open Project
		4. Look In: This Project doublecode
		5. Selected: app
		6. Click Open Project
	***** 
		Now You must have app in to the Apache NetBeans
		
		Next Step
		1. Reload Project
		2. Go to  
			app --> Source Packages [java]
			doublecode --> now you can run this program
	***** 
		run with (F6)
		run with Editor 
		If it doesn't work,  try other way to run....
	*****
		So this running
			please wait more than 20 second
